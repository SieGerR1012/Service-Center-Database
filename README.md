# Service Center Database

Ппроект реляционной базы данных для автоматизации деятельности сервисного центра.

## Возможности

- База данных позволяет хранить информацию о клиентах, заказах, оказываемых услугах, комплектующих, поставщиках, договорах и платежах
- Демонстрация проектирование нормализованной структуры данных, использование ограничений целостности

## Технологии

- PL/pgSQL
- Триггеров
